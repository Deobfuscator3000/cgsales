Introduction
------------
 This program removes SecuGen parallel driver which is installed on the system.
 Supported OS : Windows NT 4.0, Windows 2000, Windows XP, Windows 2003


Usage
-----

1. Execute the program.

2. Click "Remove" button.

3. The following messages will display by the driver state.

The driver is not running.
The driver is starting.
The driver is stopping.
The driver is running.
The driver continue is pending.
The driver pause is pending.
The driver is paused.
The driver is unknown state.

4. Reboot the system.
