５．使用方法

　★FSB変更
　　①setfsb.exeを起動する。
　　②マザーボード（パソコン）のクロックゼネレータを選択する。
　　③『Get FSB』ボタンをクリックする。
　　　必要に応じてFSB刻み(□Ultraチェックボックス)を設定する。
　　④設定する FSB(/AGP) を スライドバー か『LEFT:RIGHT』ボタンで選択する。 
　　⑤『Set FSB』ボタンをクリックする。 
　　　正常終了すれば、パソコンのクロックが変化しているはずです。 
　
　★自動耐性評価(Windows 98/Meのみ動作する機能です)
　　これはパソコンをハングアップさせて耐性評価をする超危険な機能です。
　　OSの再インストールやハードウェアに致命的なダメージ(=壊れる)を与える可能性がありますので、
　　これらが嫌な方はご使用にならないで下さい。
　　①『Auto』ボタンを有効にする。(★FSB変更①②③の操作)
　　②開始FSB『Current CPU(DDR/AGP/PCI)』をセットする。(★FSB変更④⑤の操作)
　　　※現在のFSBからスタートさせるときは必要ありません。
　　③終了FSB『Select CPU(DDR/AGP/PCI)』をスライドバー か『LEFT:RIGHT』ボタンで選択する。
　　　自動耐性評価をするときは必ずSelect FSB>Current FSBにして下さい。
　　　※Select FSB≦Current FSBのときは『Set FSB』ボタンの動作とSetFSB画面の保存をします。
　　④『Auto』ボタンをクリックする。 
　　　『Current CPU(DDR/AGP/PCI)』から『Select CPU(DDR/AGP/PCI)』まで1ステップ刻みでFSBを上げ、
　　　SetFSB画面をハードディスク(setfsb_2_2_134_98\captureフォルダ)に保存します。
　　　ファイル名は"axxx.xMHz.bmp"(xxx.xはCPUのMHz表示)になります。
　　　※captureフォルダは自動生成します。
　　　※自動耐性評価のキャンセルはプログラム終了(SetFSB画面の右上x)することで可能です。
　　⑤パソコンがフリーズする。
　　⑥パソコンを再起動させて"axxx.xMHz.bmp"によりFSB耐性を評価する。

　★画面キャプチャ
　　①『Capture』ボタンをクリックする。
　　　SetFSB画面をハードディスク(setfsb_2_2_134_98\captureフォルダ)に保存します。
　　　ファイル名はmxxx.xMHz.bmp"(xxx.xはCPUのMHz表示)になります。
　　　※captureフォルダは自動生成します。

　★WindowsのスタートアップでFSBを自動設定する。
    コマンドラインスイッチ(-x,-X,/x,/X)
	-w : Wait [00-99 sec] 省略=10sec(default)
	-s : Set FSB 000-999 [MHz]
	-i : Increment(targetFSB > currentFSB) or Decrement(targetFSB < currentFSB) [00-99 MHz] 省略(0)=max(default)
	-u : Ultra 省略(0)=normal(default) , 1=ultra
	-b : Background 省略(0)=normal(default) , 1=background
	-p : PCI-E [000-999 MHz] 省略=none
	      FSB overclock[PCI-Eを設定後にFSBを増加], FSB underclock[FSBを減少後にPCI-Eを設定]
	-q : Quiet [GUI表示無し]
	-cg[string] : Clock Generator 省略=none 最大15文字

　　①SetFSBのショートカットを作成しプロパティを開く。
　　②リンク先(T)：のsetfsb.exeの後に半角スペース+半角記号英数字でコマンドを設定する。
　　　例）Wait=5sec, Ultra, Set FSB=150MHz, Increment=5MHz
　　　C:\WINDOWS\ﾃﾞｽｸﾄｯﾌﾟ\setfsb_2_2_134_98\setfsb.exe -w05 -s300 -cg[ICS9LPRS918BKL] -u1 -i5
　　　"C:\Program Files\setfsb_2_2_134_98\setfsb.exe" -w05 -s300 -cg[ICS9LPRS918BKL] -u1 -i5
　　　※Waitを省略するとデフォルト値10secになります。
　　　※Ultraを省略するとデフォルト(標準)になります。
　　　※Incrementを省略すると初回で150MHzになります。
　　③ショートカットをスタートアップフォルダに移動する。
　　　※中止したいときは『Get FSB』又はプログラム終了(SetFSB画面の右上x)することで可能です。
　　　※コマンドライン機能の実行後、SetFSBは自動終了します。

　※エラーについて
　　・『SMBus Error』   　　　… エラーによりFSBを変更することができません。
　　・『Chipset Error』 　　　… チップセットに対応していません。
　　・『PLL ID Error』  　　　… クロックジェネレータが違います。
　　・『PLL Byte Error』　　　… クロックジェネレータが違います。
　　・『IDT Error』　　 　　　… クロックジェネレータが正常に動作しません。"CV122CPVG"
　　・『RTM Error』　　 　　　… クロックジェネレータがサポートしていない設定です。"RTM520-39D"
　　・『WinRing0 Error』　　　… ハードウェア制御DLLのエラーです。
　　・『Activation ** Error』 … ID番号をメールして活性化キーを入手してください。
